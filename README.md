# 📊 서울시 공적 노인복지 인프라 공급 불균형 분석
서울시 공공데이터를 활용하여 고령 인구 대비 복지 시설의 취약 지역을 분석한 인터랙티브 대시보드입니다.

## 🛠 사용 기술
- **Data Analysis:** Pandas, Matplotlib
- **Visualization:** Plotly, Folium, Deck.gl
- **Web Dashboard:** HTML5, CSS3, JavaScript (Chart.js)

## 📍 주요 기능
- 구별/동별 노인 복지 시설 공급지표 분석
- 3D 지도를 활용한 공공 복시시설 취약 지역 시각화
- 의료 복지 및 취약 지대 분석 리포트 제공
